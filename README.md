# 턴제 텍스트 RPG 게임

## 📖 패치 노트

### 📅 2025-10-22
#### 개발 툴 변경
- 메인 언어를 JavaScript 에서 TypeScript로 변경
- React + Vite 프레임워크 추가

### 📅 2025-10-23
#### 플레이어 공격 빗나감 보정 시스템 추가
- 캐릭터의 공격이 3회 빗나갈 경우 다음 공격은 반드시 크리티컬 판정 + 필중으로 적용
- 공격 발동시 특수 대사 출력

### 📅 2025-10-25
#### 회복 시스템 패치 
- 이제 전투 시 회복 가능 횟수가 5회로 제한됨

#### 캐릭터 밸런스 조절
- 전사 : 방어력 증가, 운 감소
- 도적 : 방어력 감소, 운 증가
- 마법사 : 공격력 대폭 증가, 운 감소

#### 상점 시스템 추가
- 이제 상점에서 장비를 구매할 수 있습니다!
- 장비는 추후 계속해서 업데이트 할 예정입니다.

### 📅 2025-11-06
#### 스킬 시스템 추가
- 이제 각 클래스 별로 스킬을 배울 수 있습니다!
- 스킬프인트는 레벨 업마다 1 포인트씩 지급됩니다.

### 📅 2025-11-07
#### 던전 추가
- 다양한 테마의 던전이 추가됩니다
- 이제 레벨에 맞는 던전에 입장해서 보다 쾌적한 레벨업이 가능합니다!

#### 무기강화 시스템 추가
- 무기강화 시스템이 추가됩니다.
- 골드를 통해 강화가 가능합니다.

#### 펫 시스템, 펫 강화 시스템 추가
- 후방에서 전투를 보조해주는 펫이 추가됩니다.
- 펫은 상점에서 골드로 구매가 가능합니다.
- 구매한 펫은 펫 강화소를 통해 강화가 가능합니다.

### 📅 2025-11-09
#### 신규 장비
- 신규 장비가 추가 되었습니다.

#### 스타터팩 추가
- 이제 캐릭터 생성 시 기본 아이템이 자동으로 지급됩니다.
- 보다 쾌적하게 게임 초반부를 진행할 수 있게 됩니다.

#### 스킬 시스템 조정 
- 스킬 레벨업이 추가됩니다. 
- 스킬은 최대 5레벨까지 레벨업이 가능하며 위력은 레벨당 20%씩 상승합니다.
- 스킬 습득 레벨을 5, 10, 15, 20으로 조정했습니다.

#### 보스 던전 추가
- 보스 던전이 추가됩니다.
- 보스 던전은 5레벨 이후부터 입장이 가능하며 클리어 시 리젠 시간이 존재합니다.
- 보스 몬스터는 높은 체력과 공격력을 가지며 몇몇 보스는 강력한 스킬을 구사합니다.

#### 네임드 몬스터 추가 
- 일반 던전에서 적을 5번 처치 시 네임드 몬스터가 등장합니다!
- 네임드 몬스터는 스탯이 일반 몬스터의 2배이며 보상 또한 좀 더 강력합니다.

### 📅 2025-11-10
#### 보스 장비 드랍 시스템 추가
- 보스 던전에서 보스 처치시 일정 확률로 장비가 드랍됩니다.
- 장비 드랍 시 장착 또는 상점 판매가 가능합니다.


### 📅 2025-11-11
#### 보스던전 버그 해결
- 전투중 보스 턴에서 보스의 행동이 정지되는 버그가 해결되었습니다.

#### 개발자용 계정 추가
- 개발자 편의를 위한 저장/로드 계정이 추가되었습니다.

#### 무기/장비/펫 등급 추가
- 등급이 추가되었습니다. (일반,희귀,영웅,전설)

#### 허수아비 추가
- 허수아비를 추가되어 데미지를 측정할 수 있습니다!

### 📅 2025-11-24
#### 보스 고유 장비 추가
- 보스 처치 시 일정 확률로 보스 고유 장비를 획득할 수 있습니다.
- 보스 고유 장비는 상점에서 구매가 불가능하며 
기존 구매 가능한 장비들보다 더 좋은 성능을 지니고 있습니다.

#### 스킬 창 추가 및 전투 메뉴 간략화
- 스킬 창을 추가하여 보다 더 스킬을 쾌적하게 사용할 수 있도록 변경했습니다.
- 스킬 창을 추가함에 따라 기존 전투 메뉴에 있던 스킬 버튼들은 스킬 창으로 이관됩니다.

## 🔧 개발 도구
- 메인 라이브러리 : React
- 개발 툴 : Vite
- 보조 프레임워크 : tailwindCSS
- 사용 언어 : TypeScript

## 🖥 실행 방법
- cd frontend
- npm run dev
